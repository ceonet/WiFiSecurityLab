## 📖 About

WiFiSecurityLab is an educational Python toolkit for studying wireless security concepts and evaluating password strength in authorized environments. It is intended for learning, research, and defensive security assessments.

---

## ✨ Features

- 📡 Wi-Fi Interface Detection
- 📶 Wireless Network Scanner
- 🔐 Password Policy Evaluation
- 📝 Logging & Statistics
- ⚡ Fast Python Implementation

---

## 🚀 Installation

```bash
git clone https://github.com/YOUR_USERNAME/WiFiSecurityLab.git
cd WiFiSecurityLab
pip install -r requirements.txt
python X-ForceWifi.py
```
